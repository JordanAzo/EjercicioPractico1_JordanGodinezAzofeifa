Bienvenidos a la Gestión de Farmacia

Este sistema permite administrar las categorías de productos de manera eficiente y segura. Puede agregar, editar y eliminar categorías para mantener el inventario actualizado.

Desarrollado con Spring Boot, Thymeleaf y Bootstrap 5, el sistema está diseñado para ser intuitivo, responsivo y adaptable a las necesidades de cualquier farmacia.
